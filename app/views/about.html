About this Project

In 2014 the City Council approved the creation of legistlation tracking system. After a competitive bid process, they awarded the bid. The system has an API that we used to create this application.

Documentation for this API can be found here.

The current City of Minneapolis website contains information on all of the proceedings of City Council, but this information is burried deep in the site and stored in PDFs.
